# tourist-destination-web
Interactive tourism website for Enrekang Regency, South Sulawesi — featuring destination map, photo gallery, travel routes, weather tips, and tour packages. Built with HTML, CSS &amp; JavaScript.

Website wisata interaktif Kabupaten Enrekang, Sulawesi Selatan — menampilkan peta destinasi, galeri foto, rute perjalanan, tips cuaca, dan paket wisata. Dibuat dengan HTML, CSS &amp; JavaScript.

This website showcases tourist destinations in Enrekang Regency, South Sulawesi. The landing page features 5 top destinations with fullscreen photos, embedded videos, and ambient sounds that match each location — wind sounds for mountain spots, water sounds for swimming areas and rivers. Visitors can explore all destinations, view an interactive map, photo gallery, travel routes from Makassar, weather tips, and available tour packages.

Website ini menampilkan destinasi wisata Kabupaten Enrekang, Sulawesi Selatan. Halaman utama menampilkan 5 destinasi terbaik dengan foto fullscreen, video, dan suara ambient yang menyesuaikan jenis wisata — suara angin untuk destinasi gunung, suara air untuk permandian dan sungai. Pengunjung dapat menjelajahi semua destinasi, melihat peta interaktif, galeri foto, rute perjalanan dari Makassar, tips cuaca, hingga paket wisata yang tersedia.
